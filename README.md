# OpenRouter for VS Code

Use any model from OpenRouter directly within VS Code Chat.

## Setup
1. Open Chat in VS Code.
2. Select an OpenRouter model from the model selector dropdown.
3. When prompted, enter your API key from [openrouter.ai](https://openrouter.ai/).

## Configuration
Use the `openrouter.modelFilter` setting to restrict which models appear in the list. 
Example: `["anthropic/claude-3.5-sonnet", "google/gemini-pro-1.5"]`

## Commands
- `OpenRouter: Reset API Key`: Use this if you need to update or change your API key.
